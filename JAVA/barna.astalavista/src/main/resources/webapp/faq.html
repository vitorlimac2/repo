Where do I get a genome assembly?
There are several alternatives for downloading genome assembly data, for example, UCSC Genome Browser and NCBI Genome FTP. For non-model organisms, you should search the data download page of the main repository of the species of your interest, for example, VectorBase, TriTryDB and others.
Where do I get a transcriptome annotation?
Usually, transcriptome annotation is available in the same repository of the genome assembly. It is important to use compatible versions of assembly and annotation.

In the UCSC Genome Browser, you can use the Table Browser to download the transcriptome annotation based on GTF (Gene Transfer Format) format. In this case, you must choose a assembly (e.g, hg19), group "Genes and Genes Predictions" and an annotation or track (e.g., GENCODE v19). The field output format convert the output to GTF selecting "GTF - gene transfer format".

In the NCBI Genome Browser, you can insert the species name in the search box and click "Search" to obtain information about the available genome assembly and annotation.
I am working with a non-model organism and I have only a transcriptome based on GFF format, how do I convert it to GTF?
If you only have a transcriptome annotation based on GFF format, convert it to GTF using gffread of Cufflinks or other script. Look other solutions in this Biostar thread: https://www.biostars.org/p/45791/.
What is the Pfam protein domain database?
According to Pfam:

"Pfam is a collection of multiple sequence alignments and profile hidden Markov models (HMMs). Each Pfam HMM represents a protein family or domain. By searching a protein sequence against the Pfam library of HMMs, you can determine which domains it carries i.e. its domain architecture. Pfam can also be used to analyse proteomes and questions of more complex domain architectures."

What is a profile-HMM?
Eddy SR (1998) says:
"Profile HMMs turn a multiple sequence alignment into a position-specific scoring system suitable for searching databases for remotely homologous sequences. Profile HMM analyses complement standard pairwise comparison methods for large-scale sequence analysis."
What is the difference between Pfam-A and Pfam-B?
Pfam-A is a manual curated Pfam entry which is built using a small number of representative sequences. Pfam manually set a threshold value for each profile-HMM and search the models against the UniProt database. All of the sequences which score above the threshold for a Pfam entry are included in the entry's full alignment.

Pfam-B contains a large number of lower quality small families automatically generated by ADDA algorithm.. It could be useful when no Pfam-A families were found. Pfam-B was discontinued as of release 28.0.
Where do I get the Pfam HMM database?
You can download different versions of Pfam from ftp://ftp.ebi.ac.uk/pub/databases/Pfam/releases/. File Pfam-A.hmm.gz contains the whole Pfam-A database.
How do I get a specific Pfam profile-HMM?
Sometimes you want only to analyse a specific subset of profile HMM from Pfam. You can search and download specifc Pfam profiles from http://pfam.xfam.org/family/browse.
